◆ context

void* AllocatorInterface_s::context

A pointer to any auxiliaty data needed by the allocator.

The purporse of this pointer is to keep track of a possible information that the allocator maybe need to know (eg. a linked list with allocated blocks). This pointer will be passed to every function defined by this interface.
